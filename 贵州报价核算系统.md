# 贵州报价—车辆公里数核算系统

> 核心算法：根据行程路线的景点排列顺序，按两套方案计算车辆总里程，自动取平均≥250km/天上限

---

## 核心算法（量化公式）

### 输入参数
```
N = 行程天数
P = 团队人数
V = 车型（5-50座，收费标准见车型价格表）
S = 住宿标准（经济/三钻/四钻/豪华）
```

### 里程输入方式
行程路线以「景点代码序列」描述，每段标注方向属性：

```
路线 = [(A→B, 方向), (B→C, 方向), ...]
方向 ∈ {往返(起点终点同), 单程(只去不回), 绕道+XXkm(匝道)}
```

---

### 方案A — 文档参考值法

直接从`贵州常规公里数.md`中提取路线参考值：

```
总里程 = Σ(文档参考值)
```

文档没有的路线，用最近似已有路线 + 差值估算。

**适用场景**：客户接受参考值、无需精确核算、标准动线

---

### 方案B — 高速系数法

```
主线（起点→最远景点→终点）= 高速地图里程 × 系数 × 方向倍数

中途景点匝道 = 距主线最近互通口里程 × 系数 × 方向倍数

方向倍数：
  顺路方向（往下一目的地方向延伸）→ ×1（单程）
  逆向方向（往回走/绕道）→ ×2（往返）
  住宿绕路 → 住宿地距下一景点 ×2（往返）

总里程 = Σ(各段计算里程) + 接机/送机里程

系数选择规则：
  1.35 = 简单路线（高速直达占比>80%）
  1.40 = 标准路线（含部分国道/省道）
  1.45 = 复杂路线（多山路/弯道）
  1.50 = 极端路线（盘山路/县道为主）
```

---

### 取足规则

```
每天平均 = 总里程 ÷ N

if 每天平均 < 250:
    计费里程 = 250 × N（按250取足）
else:
    计费里程 = 总里程
```

## 报价分项公式

```
门票A = Σ(景区全含价 × P)
  注：门票价格必须从上方「门票基准价」表取数，不得推算或记忆

车费B = 计费里程 × 车型单价
  (含油费、路桥费、空返费、司机食宿补贴)

住宿C = Σ(各晚的间数 × 该晚单价)
  间数 = ceil(P / 2)（2人/间，单男单女另算）
  司机导游房间 = 司陪住（免费协调）或单独核算

用餐D = 餐标价 × 正餐数量 × P
  (早餐由酒店含，不加收)

导服E = 导游日薪 × N
  (普通300-400元/日，中级400-500元/日，高级600-800元/日)

司陪住宿F = 司机住宿(间×晚×价) + 导游住宿(间×晚×价)
  (与客人同酒店，选最低房价；贵阳段一般不安排)

操作费G = 15-30元/人 × P
  (线路设计、调度、发票税点)

总价 = A + B + C + D + E + F + G
人均 = 总价 ÷ P
```

## 路线分段核算示例（方案B 标准）

以 `贵阳接→黄果树→天眼→小七孔→西江→青岩送` 为例：

### 主线分析
```
最远点：西江（从贵阳算约230km）
主线：贵阳→黄果树→天眼→荔波→西江→青岩
```

### 各段核算

| 段 | 方向 | 地图km | ×系数 | 方向倍数 | 小计 |
|----|:---:|:------:|:-----:|:--------:|:---:|
| D1 接机 | 单程 | 30 | 1.4 | ×1 | 42 |
| D2 贵阳→黄果树→住安顺 | 顺路单边 | 130 | 1.4 | ×1 | 182 |
| D2 安顺→天眼→住荔波 | 顺路单边 | 230 | 1.4 | ×1 | 322 |
| D3 荔波→小七孔（已到） | 住地已到 | — | — | — | 0 |
| D4 荔波→西江 | 顺路单边 | 200 | 1.4 | ×1 | 280 |
| D5 西江→青岩→送机 | 顺路单边 | 270 | 1.4 | ×1 | 378 |
| **合计** | | **860** | | | **1,204** |

### 取足校验
```
1,204km ÷ 5天(带客)=240km/天 < 250
→ 取足：250km × 6天(含车日) = 1,500km（方案A结果）
或 继续走方案B路线 → 1,204km直接使用
```

> 注意：用户已确认 GZ20260502-02 采用方案A=1,500km、方案B=1,652km (系数×1.4)
> 1,652 = 860×1.4 后再加绕道 + 再取足？需复核

## 路线方向判定速查表

| 行程方向 | 方向判定 | 倍数 |
|----------|:-------:|:----:|
| A→B→C→A（回到起点） | 往返全路线 | ×1 |
| A→B→C→...（不回起点） | 顺路单边各段 | ×1 |
| 中途去D再回主路线 | 逆向→往返 | ×2 |
| A→B→A（半日游返回） | 往返 | ×2(实际为往返，但已在主线中) |
| 住X地，次日从X出发 | 住宿绕路 | 接上次结束点→X + X→次日首点 |

## 常见错误记录

| 错误场景 | 错误做法 | 正确做法 |
|----------|---------|----------|
| 绕道方向判定 | 所有绕道都×2 | 顺路绕道×1，逆向绕道×2 |
| 接机/送机计算 | 加入总里程取足 | 单列不计入取足（接机是服务，不计行车日） |
| 司机住宿天数 | 按N天全算 | 客人住哪天司机就住哪天，回程当天不算 |
| 系数选择 | 一律1.4 | 按高速占比选择1.35-1.50 |
